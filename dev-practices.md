# Development Practices

**Environment Management:**
- Default to conda/mamba environments
- Use mambaforge as standard
- Avoid pip-only workflows

**Documentation:**
- Fetch official docs for xarray, pandas, matplotlib rather than using recalled patterns
- Documentation-first approach for core libraries

**Code Quality:**
- Target Python 3.10+
- Answer directly about version-specific behavior when uncertain
- Flag genuinely uncertain areas briefly, then provide best answer
